About the Research Platform

CB2 is also a mature, flexible research and data collection platform made to study natural language instructions grounded in a 3D environment.

Diverse Map Generation Algorithm

Each game takes place in a unique, diverse world thanks to our map generation algorithm

Real-world experience

The platform has been used for 500+ games on Amazon mechanical turk, and we've integrated user testing & feedback into making the UI, platform, and data collection process as smooth as possible.

Google account authentication is also supported for public data collection.

Made for serving models

The platform makes it easy to run human-human, human-bot, or even bot-bot games.

The CB2 Python API makes it extremely simple to develop your own bot to interface with CB2.

We also have an OpenAI gym environment.

Simulated Self-play

The game logic has been optimized to run simulated self-play locally, capable of running approximately 1 game per second per CPU

Map Editor and Custom Scenario Support

Custom maps and scenarios can be crafted to support any research needs.

About the game

CB2 is a collaborative, multiplayer game.

Collect sets of cards to score points

Each set consists of 3 unique cards.

The cards cannot have the same color, shape, or count.

Work with your partner to score higher!

You play as either the leader or the follower. Combine your unique advantages to improve your score.

The leader can see the entire board all at once. The follower can move much farther per turn.

Make the most of limited time!

The game ends when you run out of turns.

You are gifted extra turns with each point you score. Turns are ended automatically when you run out of time, so keep an eye on the clock!
